# 😄 Moodly

Moodly adalah aplikasi yang dirancang untuk membantu pengguna mencatat dan memantau suasana hati secara sederhana, cepat, dan intuitif.

---

## 🌐 Live Demo

👉 https://anandadavidsaputra.github.io/Moodly/

---

## 🧩 Overview

Moodly dibangun sebagai proyek latihan pengembangan web untuk:

* mencatat suasana hati harian
* menyimpan catatan refleksi singkat
* menampilkan riwayat mood
* menampilkan statistik mood dalam bentuk visual

Aplikasi ini dibuat sebagai sarana eksplorasi dan latihan dalam membangun fitur interaktif menggunakan JavaScript serta pengelolaan data berbasis localStorage.
